# Web Development Basics – Task 1

## Project Title

**Task 1: Create a Simple Webpage Using HTML, CSS, and JavaScript**

## Timeline

9 Days

## Introduction

This repository contains **Task 1** of the Web Development Basics program.
The objective of this task is to understand the foundational concepts of **HTML, CSS, and JavaScript** and apply them to build a simple interactive webpage.

## Objective

The main objective of this task is to learn the fundamentals of web development by:

* Creating the structure of a webpage using HTML
* Styling the webpage using CSS
* Adding basic interactivity using JavaScript

## Task Description

In this task, a basic webpage was created that includes headings, paragraphs, images, and hyperlinks.
The webpage was styled using CSS to improve its appearance by adding colors, fonts, spacing, and layout adjustments.
JavaScript was used to add simple interaction by implementing a button that displays an alert message when clicked.

## Technologies Used

* HTML5
* CSS3
* JavaScript

## Features Implemented

* Basic webpage structure using HTML
* Headings and paragraph elements
* Displaying an image on the webpage
* Adding hyperlinks to external resources
* Styling with colors, fonts, and spacing using CSS
* Interactive button using JavaScript alert function

## How to Run the Project

1. Download or clone this repository.
2. Open the project folder in Visual Studio Code.
3. Open the **index.html** file in a web browser.
4. The webpage will load and display the content.
5. Click the button on the webpage to see the alert message generated by JavaScript.
